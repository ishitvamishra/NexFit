# 🏋️ NexFit – AI Powered Fitness & Nutrition Companion

NexFit is an AI-driven fitness and nutrition tracking Android application designed to help users maintain a healthy lifestyle through smart workout tracking, calorie monitoring, progress analytics, and personalized fitness guidance.

Built as a solution aligned with **Smart India Hackathon (SIH25129) – Fitness & Sports**, the application focuses on improving fitness engagement using intelligent tracking and gamification features.

---

# 🚀 Features

## 🧠 Smart Dashboard
- Track daily steps, calories burned, and activity progress
- Quick access to workouts, meals, and fitness goals

## 🏋️ Workout Logger
- Log exercises, sets, reps, and weights easily
- Monitor workout consistency and progression

## 📈 Progress Analytics
- Interactive charts for workout and fitness progress
- Track improvements over time with visual insights

## 🥗 Diet Planner
- Monitor calorie intake and macronutrients
- Personalized meal suggestions based on goals

## 🔔 Smart Reminders
- Notifications for workouts, meals, and hydration
- Helps users stay consistent with fitness routines

## 🏆 Gamification
- Fitness streaks and achievement tracking
- Community challenges and leaderboards

---

# 🛠️ Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose
- **Architecture:** MVVM
- **Database:** Firebase / Room Database
- **Tools:** Android Studio, Git, GitHub
